* 查看面板入口（也可以通过该命令判断服务器是否安装了宝塔面板）

  ```sh
  /etc/init.d/bt default
  ```

  

* 使用 bt 命令 重置用户密码

  ![f39a46610138cf74d3687a6cf8be9f20](./宝塔常用指令/images/f39a46610138cf74d3687a6cf8be9f20.png)

* 重启面板

  ```sh
  bt restart
  ```
